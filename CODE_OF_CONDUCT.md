# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and
demeaning conduct are not tolerated. We welcome contributors of all backgrounds
and experience levels.

The full text is available at
https://www.contributor-covenant.org/version/2/1/code_of_conduct/.

To report unacceptable behavior, contact the maintainers via a private GitHub
Security Advisory or the contact listed in the repository's community profile.
Reports will be handled confidentially.
